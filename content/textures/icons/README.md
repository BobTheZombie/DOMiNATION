# RTS UI Icons

## Resource Icons
Files in `resources/` are 128x128 PNG icons with transparent backgrounds and simple high-contrast silhouettes designed to scale down to 32px HUD use.

- `food_resource_icon_128.png`
- `wood_resource_icon_128.png`
- `metal_resource_icon_128.png`
- `wealth_resource_icon_128.png`
- `knowledge_resource_icon_128.png`
- `oil_resource_icon_128.png`

Generated via `python3 tools/generate_rts_icons.py`.
